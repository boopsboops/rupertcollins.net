Figure S2 (online version). NJ phylogram (reduced RHO data) generated in phyloXML SVG (scalable vector graphic) format. Data including identifiers, sequences, trace files, museum voucher codes and specimen images are accessed via the BOLD and GenBank Web sites using URLs embedded in the taxon names. This figure is best viewed with Mozilla Firefox to fully enjoy the benefits of SVG and URL linking. May take up to one minute to load. A scripting "error" may appear in some browsers—this is the browser taking time to render the complex diagram. Phylogram can be saved as a pdf by printing to file using a custom paper size (approximately 750 mm height). Links can be opened in a new tab using Ctrl+LeftClick. Archived version found here.
